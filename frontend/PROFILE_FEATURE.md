# Страница профиля пользователя

## Что добавлено

### 1. Страница профиля (`/profile`)
Страница доступна только для авторизованных пользователей.

### 2. Две вкладки

#### **Данные профиля**
- Просмотр данных: имя, email, роль
- Редактирование имени и email
- Сохранение изменений (обновление в localStorage и контексте)

#### **Смена пароля**
- Ввод текущего пароля
- Ввод нового пароля (минимум 8 символов)
- Подтверждение пароля
- Модальное окно для ввода кода подтверждения (6 цифр)
- Код отправляется на email пользователя

### 3. API методы

#### User API
```javascript
userAPI.update(id, data)        // Обновить данные пользователя
userAPI.getById(id)             // Получить данные пользователя
```

#### Password API
```javascript
passwordAPI.change(data)        // Запрос на смену пароля
passwordAPI.confirmChange(data) // Подтверждение смены пароля
```

### 4. Backend endpoints

- `PUT /users/{id}` - обновление данных пользователя
- `POST /password/change` - запрос смены пароля
- `POST /password/change/confirm` - подтверждение смены пароля

### 5. Процесс смены пароля

1. Пользователь вводит текущий пароль, новый пароль и подтверждение
2. На email отправляется 6-значный код подтверждения
3. Пользователь вводит код в модальном окне
4. Пароль обновляется в базе данных

### 6. Design

- Табы для переключения между разделами
- Красивые формы с валидацией
- Модальное окно для подтверждения смены пароля
- Адаптивный дизайн
- Индикаторы загрузки

## Использование

1. Войдите в систему
2. Перейдите в "Профиль" из навигации
3. Выберите вкладку:
   - "Данные профиля" - для редактирования информации
   - "Смена пароля" - для изменения пароля

## Особенности

- Роль пользователя отображается только для чтения
- Email и имя можно изменить
- При смене пароля требуется подтверждение по email
- Все изменения сохраняются в localStorage и обновляют контекст приложения
